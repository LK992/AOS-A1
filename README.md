In this repository there are three bash scripts and one python script.

To run the bash you can use WSL (Windows Linux Subsystem) and it should work perfectly if file order stays the same, Otherwie can be run in a virtual machine using any Linux Distro.

To run the python script ensure that terminal is in the correct directory to ensure the python script can see the required files, for example the terminal should be in directory /task3 before running.
